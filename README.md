# mtgconv

Magic the Gathering Deck list converter.

Program and framework to convert deck lists between different input sources and output formats.

## Usage

Convert a Moxfield deck to .dck decklist format

```bash
$ ./mtgconv --user-agent "$MOXKEY" https://moxfield.com/decks/Wrcumkgcc0qjIB2bwoDvqQ
```
